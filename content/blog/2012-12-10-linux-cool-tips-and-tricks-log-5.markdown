## Find if your system is 32 or 64 bits

## Find if a binary is compiled for 32 or 64 bits






file -be elf _binary-path_

Example
file -be elf /usr/bin/sudo

setuid **ELF 64-bit** LSB executable, x86-64, version 1 (SYSV)
